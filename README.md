## Integración de módulos

El software fue desarrollado bajo una arquitectura modular, en la cual sus funciones principales se dividen en distintos módulos independientes. Cada módulo cumple una función específica, como la gestión de usuarios, el manejo del menú virtual y la calificación del servicio.

La integración de estos módulos permite que el sistema opere de manera conjunta y eficiente, facilitando el intercambio de información entre ellos. Esta estructura mejora la organización del software, simplifica su mantenimiento y permite futuras actualizaciones sin afectar el funcionamiento general del sistema.
